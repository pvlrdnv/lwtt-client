# Переименование файлов

Начиная с версии 4.15 файлы используют короткий префикс `lwtt`.

## Новые имена

```text
lwtt_tray_start.bat
lwtt_tray.ps1
lwtt_manager.ps1
lwtt_manager_start.bat
lwtt_common.ps1
lwtt_start.bat
lwtt_stop.bat
lwtt_runner.ps1
lwtt_test_worker.ps1
lwtt_autostart_install.bat
lwtt_autostart_remove.bat
lwtt_client.toml
```

## Старые имена

Старые файлы с префиксом `likeweb_` не нужны для новой версии и могут быть удалены после проверки работоспособности.

## Важно

Не удаляйте папку `profiles`, если хотите сохранить уже добавленные серверы.
