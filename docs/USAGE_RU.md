# Использование LW TrustTunnel Client

## Запуск

Запустите:

```text
lwtt_tray_start.bat
```

После запуска возле часов появится значок LW TrustTunnel Client.

## Состояния значка

- `icons8-start-96.ico` — соединение установлено.
- `icons8-stop2-96.ico` — соединение отсутствует.
- `icons8-waiting-96.ico` — идет подключение или отключение.

## Основное меню

Меню открывается кликом по значку возле часов.

Основные пункты:

- **Статус** — текущее состояние.
- **Подключить** — подключить выбранный сервер.
- **Отключить** — остановить текущее подключение.
- **Управление серверами** — открыть список сохраненных серверов.
- **Добавить сервер** — добавить новый сервер.
- **Сохранить журнал диагностики** — создать диагностический `.txt`.
- **Открыть папку программы** — открыть папку с файлами.
- **Закрыть приложение** — остановить подключение и закрыть приложение.

## Управление серверами

В окне управления можно:

- подключить выбранный сервер;
- отключить текущее подключение;
- добавить новый сервер;
- редактировать сохраненный сервер;
- открыть файл профиля;
- удалить профиль.

## Диагностика

Если что-то не работает, используйте пункт меню:

```text
Сохранить журнал диагностики...
```

Файл диагностики можно отправить администратору. В нем указывается версия приложения, состояние TrustTunnel, активный сервер и последние строки журналов.
