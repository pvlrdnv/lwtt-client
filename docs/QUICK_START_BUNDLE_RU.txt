LW TrustTunnel Client — очень краткая инструкция
================================================

1. Скачайте готовый архив для Windows x86_64:

https://github.com/pvlrdnv/lwtt-client/raw/main/dist/LWTT_Client_Bundle_windows_x86_64.zip

После нажатия на ссылку скачивание должно начаться сразу.

2. Распакуйте архив в папку:

C:\Trusttunnel

3. Запустите файл:

C:\Trusttunnel\lwtt_tray_start.bat

Если Windows спросит разрешение администратора, нажмите «Да».

4. Получите в тг-боте настройки сервера и файл сертификата .pem, если он требуется.

5. Нажмите значок LWTT возле часов и выберите «Добавить сервер».

6. Вставьте целиком сообщение из тг-бота с настройками сервера.

7. Выберите файл сертификата .pem, если он был прислан.

8. Нажмите «Сохранить и подключить».

Если непонятно, откройте полную инструкцию:

https://github.com/pvlrdnv/lwtt-client/blob/main/docs/INSTALL_RU.md
