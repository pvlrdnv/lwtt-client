# Безопасность и приватность

## Что хранится локально

LW TrustTunnel Client хранит настройки серверов локально на компьютере пользователя.

В папке `profiles` могут находиться:

- адреса серверов;
- имена пользователей;
- пароли;
- параметры сертификатов;
- локальные копии `.pem` сертификатов.

## Что не должно попадать в GitHub

В репозитории и публичных архивах не должно быть:

```text
profiles/
profiles/certificates/
profiles/backups/
*.pem
lwtt_client.toml
log/
LW_TrustTunnel_Client_diagnostic_*.txt
```

## Диагностические файлы

Диагностические файлы помогают разбирать ошибки, но могут содержать техническую информацию о компьютере и сервере. Перед публикацией в открытом доступе их нужно проверять и очищать.

## Права администратора

Права администратора нужны для запуска TrustTunnel и работы сетевого адаптера Wintun. Не запускайте приложение из непроверенных источников.
