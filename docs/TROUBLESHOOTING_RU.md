# Диагностика и частые проблемы

## Приложение не запускается

Проверьте, что все файлы лежат в одной папке:

```text
trusttunnel_client.exe
wintun.dll
lwtt_tray_start.bat
lwtt_tray.ps1
lwtt_common.ps1
```

Запустите `lwtt_tray_start.bat` обычным двойным кликом и подтвердите запрос прав администратора.

## Соединение не устанавливается

Проверьте:

- правильно ли выбран сервер;
- правильно ли введены адрес, имя пользователя и пароль;
- добавлен ли `.pem` сертификат, если он нужен серверу;
- не блокирует ли запуск Защитник Windows или антивирус;
- установлен ли правильный архив TrustTunnel под архитектуру Windows.

## TrustTunnel запустился, но интернет не изменился

Проверьте внешний IP через любой сервис проверки IP. Если соединение работает, внешний IP должен отличаться от обычного домашнего адреса.

## Ошибки Wintun

Иногда Windows задерживает создание или освобождение Wintun-адаптера. Обычно помогает:

1. отключить соединение;
2. закрыть LW TrustTunnel Client;
3. перезагрузить Windows;
4. снова запустить `lwtt_tray_start.bat`.

## Как отправить отчет администратору

В меню трея выберите:

```text
Сохранить журнал диагностики...
```

Отправьте созданный `.txt` файл администратору. Не публикуйте диагностические файлы в открытом доступе.
