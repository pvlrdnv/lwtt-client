LW TrustTunnel Client v4.15
===========================

Назначение
----------
LW TrustTunnel Client управляет сохраненными профилями TrustTunnel, переключением серверов, значком в трее, проверкой сервера и диагностикой.

Основные файлы
--------------
- lwtt_tray_start.bat — запуск приложения и значка в трее.
- lwtt_tray.ps1 — логика трея и меню.
- lwtt_manager.ps1 — окно управления серверами.
- lwtt_manager_start.bat — прямой запуск окна управления.
- lwtt_start.bat — запуск TrustTunnel с активной конфигурацией.
- lwtt_stop.bat — остановка TrustTunnel.
- lwtt_runner.ps1 — технический запуск TrustTunnel и запись журналов.
- lwtt_common.ps1 — общие функции приложения.
- lwtt_test_worker.ps1 — проверка сервера из окна добавления/редактирования.
- lwtt_client.toml — активная конфигурация TrustTunnel, создается автоматически из выбранного профиля.

Папки
-----
- profiles — сохраненные профили серверов. Не удаляйте при обновлении.
- profiles\certificates — сертификаты профилей.
- profiles\backups — резервные копии активной конфигурации.
- log\client — журналы запуска TrustTunnel.
- log — журналы приложения и служебное состояние.

Обновление с предыдущей версии
------------------------------
1. Закройте приложение через меню трея.
2. Распакуйте архив v4.15.
3. Скопируйте файлы в папку программы с заменой.
4. Папку profiles не удаляйте.
5. Запустите lwtt_tray_start.bat.
6. Если используется автозапуск, повторно запустите lwtt_autostart_install.bat.

Примечание по переименованию
----------------------------
В v4.15 файлы приложения переименованы в соответствии с названием LW TrustTunnel Client и коротким префиксом LWTT.
Старые файлы предыдущих сборок больше не используются приложением.

Диагностика
-----------
Диагностический файл создается из меню трея: «Сохранить журнал диагностики...».
В диагностике указывается версия приложения и последние журналы TrustTunnel.
