LW TrustTunnel Client — структура v4.16
========================================

Для обычного пользователя в корне папки установки должен быть только основной файл запуска:

lwtt_tray_start.bat

Все технические файлы приложения находятся во вложенной папке:

lwtt_app\

Внутри lwtt_app находятся TrustTunnel, Wintun, скрипты LWTT, иконки, профили, журналы и активная конфигурация.

Быстрый запуск:
1. Распакуйте готовый bundle-архив в C:\Trusttunnel.
2. Запустите C:\Trusttunnel\lwtt_tray_start.bat.
3. Получите в тг-боте настройки сервера и файл сертификата .pem, если он требуется.
4. В меню LWTT выберите «Добавить сервер».
5. Вставьте целиком сообщение из тг-бота с настройками сервера.
6. Выберите файл сертификата .pem, если он был прислан.
7. Нажмите «Сохранить и подключить».

Не переносите вручную файлы из папки lwtt_app в корень.
